# 📝 To-Do List Web Application

A full-featured To-Do List web application built using **Flask**, **SQLite**, **HTML**, **CSS**, and **JavaScript**.  
This project allows users to manage tasks efficiently with priorities, due dates, search, sorting, and real-time dashboard statistics.

## 🚀 Features

- Add tasks with:
  - Title
  - Due Date
  - Priority (High / Medium / Low)

- Task Management:
  - Edit tasks
  - Delete tasks
  - Mark tasks as completed
  - Overdue task highlighting

- Sorting:
  - Sort by Due Date
  - Sort by Priority
  - Newest First
  - Oldest First

- Search:
  - Search tasks instantly

- Dashboard Statistics:
  - Total Tasks
  - Completed Tasks
  - Pending Tasks
  - Overdue Tasks
  - Priority Distribution

- Data Visualization:
  - Pie chart using Chart.js

- Responsive Design:
  - Works on desktop and mobile
  - Media queries implemented

## ⚙ Installation and Setup

Step 1 — Clone the repository -- git clone https://github.com/NIDHI-2007/To_Do_List_App/
Step 2 — Navigate to the project -- cd To_Do_List_App
Step 3 — Install dependencies --  pip install flask
                                  pip install flask-sqlalchemy
Step 4 — Run the application -- python app.py
Step 5 — Open in browser -- http://127.0.0.1:5000
